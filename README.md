## Description

Backend repository for Bizresolve system application.

## Installation

```bash
#using npm
$ npm install 

#using yarn
$ yarn install 
```

## Connecting to the database

### Setting up docker
Make sure you have docker and docker compose installed on your local machine. You can use docker desktop for ease of use.
Run 
```bash
docker compose up dev-db -d
```

on the root folder of the application to create a docker container in which the database will be run

### setting up the database on docker
Once the docker container containing postgresql is up and running,
Get the container ID of the specific container by running 
```bash
docker ps
```
Get access to the container by running 
```bash
docker exec -it <containerID> bash
```

Once logged into the container, create a database user with your prefered username and password
Then create an empty database called biz_resolve
Edit the .env file with the created username and password
Make sure the database name is consistent.
```bash
#This should guide you on how to setup the database connection string
postgresql://<username>:<password>@localhost:5440/biz_resolve?schema=public
```



## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
